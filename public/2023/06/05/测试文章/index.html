新建文章

npx hexo new "文章标题"
如果在本地直接添加 markdown 格式文件，则 http://localhost:4000/ 这个地址中会更新文章，但是 GitHub 地址中不会更新文章。

新建页

清除缓存

npx hexo clean

生成

npx hexo g

部署到远程服务器（此处部署到GitHub库）

npx hexo deploy

!!!本地启动!!!

npx hexo s

模板配置文件地址

C:\Users\用户名\博客所在的文件夹\themes\next
